                              법인카드사용지침
================================================================================

1. 개요
-------
본 지침은 법인카드의 발급, 사용, 관리에 관한 실무 절차를 안내합니다.

================================================================================

2. 법인카드 발급
---------------

2.1 발급 대상
- 정규직 대리급 이상
- 입사 3개월 경과 후
- 계약직: 부서장 추천 시 발급 가능

2.2 발급 신청
① 절차:
   1) 사내 포털 > 총무/재무 > 법인카드 신청
   2) 신청서 작성 (사용 목적, 예상 사용액)
   3) 부서장 승인
   4) 재무팀 검토
   5) 카드 발급 (약 7영업일)

② 수령:
   - 본인 직접 수령 (재무팀)
   - 신분증 지참 필수
   - 카드 수령 서약서 서명

2.3 발급 한도
| 직급 | 월 한도 | 1회 한도 |
|------|---------|----------|
| 임원 | 1,000만원 | 500만원 |
| 부장 | 500만원 | 200만원 |
| 차장/과장 | 300만원 | 100만원 |
| 대리 | 200만원 | 50만원 |

2.4 한도 조정
- 일시 한도 증액: 부서장 + 재무팀장 승인
- 영구 한도 변경: CFO 승인
- 신청: 사내 포털 > 법인카드 > 한도 조정 신청

================================================================================

3. 법인카드 사용
---------------

3.1 사용 가능 범위
✓ 업무용 식대 (회의비, 접대비)
✓ 출장 경비 (교통, 숙박, 식비)
✓ 업무용 물품 구매
✓ 교육/세미나 등록비
✓ 업무 관련 택시비

3.2 사용 금지 범위
✗ 개인 용도 사용 (절대 금지)
✗ 현금 서비스/현금 인출
✗ 유흥업소 (단란주점, 룸살롱, 노래방)
✗ 귀금속, 명품 구매
✗ 상품권, 기프트카드 구매
✗ 온라인 게임, 도박 사이트
✗ 개인 쇼핑 (의류, 전자제품 등)
✗ 가족/지인 대리 결제

3.3 사용 시간
- 일반 사용: 업무 시간 내 (주중 08:00~22:00)
- 야간/주말 사용: 업무상 불가피한 경우 (사유 필수 기재)

3.4 해외 사용
- 해외 출장 시 사용 가능
- 해외 사용 한도: 국내 한도와 동일
- 환율: 카드사 결제일 기준 환율

================================================================================

4. 정산
-------

4.1 정산 기한
- 사용 후 10영업일 이내

4.2 정산 방법
① 경비정산시스템 접속
② [법인카드정산] 메뉴
③ 미정산 내역 확인
④ 사용 용도 입력
⑤ 세부 내용 기재
⑥ [제출]

4.3 미정산 시 불이익
- 15일 미정산: 경고 메일
- 30일 미정산: 카드 일시 정지
- 60일 미정산: 급여 공제

================================================================================

5. 카드 관리
-----------

5.1 보관
- 본인만 사용 (타인 대여 금지)
- 분실 주의
- 카드 번호 노출 금지

5.2 분실/도난 시
① 즉시 카드사 분실신고 (1577-XXXX, 24시간)
② 재무팀에 분실 보고 (다음 영업일까지)
③ 분실 경위서 제출
④ 재발급 신청

5.3 재발급
- 분실/훼손 시 재발급 신청
- 재발급 비용: 회사 부담
- 2회 이상 분실: 재발급 불가 검토

5.4 반납
- 퇴직, 전배, 보직 변경 시 반납
- 반납 시점: 발령일 또는 퇴직일
- 반납 장소: 재무팀
- 미정산 건 완료 후 반납

================================================================================

6. 위반 시 조치
--------------

6.1 위반 유형별 조치
| 위반 유형 | 1회차 | 2회차 | 3회차 이상 |
|-----------|-------|-------|------------|
| 개인 사용 | 환수+경고 | 환수+카드회수+감봉 | 해고 검토 |
| 금지 장소 사용 | 환수+경고 | 환수+카드회수 | 해고 검토 |
| 타인 대여 | 경고 | 카드 회수 | 징계 |
| 정산 지연 | 경고 | 카드 일시정지 | 카드 회수 |

6.2 부정 사용 발견 시
- 해당 금액 전액 환수
- 인사규정에 따른 징계
- 중대 사안 시 형사고발 (업무상 횡령)

7. 모니터링
----------

7.1 상시 모니터링 대상
- 금지 업종 사용
- 야간(22시 이후) 사용

7.2 소명 요청

- 3영업일 내 소명서 제출
- 미제출 또는 불충분 시 조사 진행

8. 자주 묻는 질문 (FAQ)
----------------------

Q1. 법인카드로 온라인 쇼핑 가능한가요?
A1. 업무용 물품 구매만 가능합니다.
    개인 쇼핑(의류, 전자제품 등)은 불가합니다.
    업무용 소프트웨어, 도서, 교육 결제는 가능합니다.

Q2. 해외 직구 결제 가능한가요?
A2. 업무 목적 구매에 한해 가능합니다.
    해외 결제 시 카드사에 따라 차단될 수 있으니
    사전에 재무팀에 문의하세요.

Q3. 카드 한도가 부족합니다.
A3. 사내 포털 > 법인카드 > 한도 조정 신청
    부서장 승인 후 재무팀에서 처리합니다.
    일시적 증액은 당일 처리 가능합니다.

Q4. 가맹점에서 결제가 거절됩니다.
A4. 확인 사항:
    - 한도 초과 여부
    - 해당 업종 사용 차단 여부
    - 카드 정지 여부
    재무팀(내선 1235)으로 연락하세요.

Q5. 부서 공용 법인카드가 있나요?
A5. 원칙적으로 개인 카드만 발급합니다.
    특수한 경우(공용 차량 유류비 등)는
    부서장 명의로 부서카드 발급이 가능합니다.
