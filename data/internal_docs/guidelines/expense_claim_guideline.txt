                              경비정산지침
================================================================================

1. 개요
-------
본 지침은 경비관리규정에 따른 경비정산의 실무 절차와 세부 기준을 안내합니다.

================================================================================

2. 정산 시스템 접속
------------------
2.1 접속 방법
- 사내 포털 > 업무시스템 > 경비정산시스템
- 또는 직접 URL: https://expense.company.com

2.2 로그인
- 사번/비밀번호 (통합 SSO)
- 모바일 앱: iOS/Android "사내경비" 앱

================================================================================

3. 경비 종류별 정산 방법
-----------------------

3.1 법인카드 정산
-----------------
① 사용 내역 자동 연동 (카드사 → 시스템)
② 정산 절차:
   1) 경비정산시스템 접속
   2) [법인카드정산] 메뉴 클릭
   3) 미정산 내역 확인
   4) 사용 용도 선택 (회의비, 접대비, 교통비 등)
   5) 세부 내용 입력 (참석자, 목적 등)
   6) [제출] 클릭

③ 사용용도 코드:
   - 100: 회의비
   - 200: 접대비
   - 300: 교통비
   - 400: 출장비
   - 500: 교육비
   - 600: 소모품비
   - 900: 기타

④ 추가 정보 입력 (필수):
   - 회의비/접대비: 참석자 명단, 목적
   - 교통비: 출발지-도착지, 목적
   - 식대: 식사 유형 (조식/중식/석식/야근)

3.2 개인경비 정산
-----------------
① 정산 절차:
   1) 경비정산시스템 접속
   2) [개인경비정산] 메뉴 클릭
   3) [신규 신청] 버튼 클릭
   4) 경비 정보 입력
      - 사용일자
      - 경비 유형
      - 금액
      - 사용 내용
   5) 영수증 첨부 (스캔 또는 촬영)
   6) [제출] 클릭

② 영수증 첨부 요건:
   - 파일 형식: JPG, PNG, PDF
   - 파일 크기: 5MB 이하
   - 영수증 필수 정보 식별 가능
   - 금액, 일자, 가맹점명 명확

③ 지급 계좌:
   - 급여 계좌로 자동 지급
   - 다른 계좌 희망 시: 인사팀에 계좌 변경 신청

3.3 출장비 정산
---------------
① 별도 "출장정산지침" 참조
② 출장정산서 + 출장보고서 제출 필수
③ 경비정산시스템 > [출장정산] 메뉴

4. 증빙 기준
-----------

4.1 적격 증빙
- 세금계산서
- 신용카드 매출전표
- 현금영수증 (지출증빙용)

4.2 간이영수증 사용 가능 조건
- 3만원 이하
- 적격 증빙 수취 불가한 경우 (농어촌 등)
- 해외 출장 시 현지 영수증

4.3 증빙 불인정 사례
- 금액, 일자 식별 불가
- 위변조 의심
- 업무 관련성 소명 불가
- 가족/지인 명의 영수증

5. 금액별 승인 체계
------------------
| 금액 구간 | 승인권자 | 비고 |
|-----------|----------|------|
| 50만원 미만 | 팀장 | 결재 1단계 |
| 50만원~100만원 | 부서장 | 결재 2단계 |
| 100만원~500만원 | 본부장 | 결재 3단계 |
| 500만원 이상 | CFO | 결재 4단계 |

================================================================================

6. 정산 기한
-----------

6.1 정산 기한
- 법인카드: 사용 후 10영업일 이내
- 개인경비: 사용 후 10영업일 이내
- 출장비: 출장 종료 후 7영업일 이내

6.2 기한 초과 시
- 15일 초과: 독촉 메일 발송
- 30일 초과: 법인카드 일시 정지 + 부서장 통보
- 60일 초과: 급여 공제 처리

7. 자주 묻는 질문 (FAQ)
----------------------

Q1. 영수증을 분실했습니다. 어떻게 하나요?
A1. 가맹점에 재발급 요청하거나, 카드사에서 이용내역서를 발급받아 제출합니다.
    재발급 불가 시 "영수증 분실 확인서" 작성 후 부서장 승인을 받으면 됩니다.
    (연 3회 한도, 건당 10만원 이하)

Q2. 법인카드 결제가 안 됩니다.
A2. 다음 사항을 확인하세요:
    - 한도 초과 여부 (시스템에서 확인)
    - 미정산 누적으로 인한 정지
    - 카드 분실 신고 여부
    문제 지속 시 재무팀(내선 1234)으로 연락주세요.

Q3. 접대비와 회의비 구분은?
A3. 
    - 접대비: 외부인(고객, 협력사)과 식사
    - 회의비: 당사 임직원만 식사
    외부인이 1명이라도 있으면 접대비입니다.

Q4. 주말 사용 건도 정산 가능한가요?
A4. 업무상 불가피한 경우 가능합니다.
    주말/공휴일 사용 건은 사유 기재가 필수이며,
    정기 모니터링 대상입니다.

Q5. 정산이 반려되었습니다.
A5. 반려 사유를 확인 후:
    - 증빙 미비: 영수증 재첨부
    - 내용 미비: 내용 보완
    - 예산 초과: 예산 조정 후 재신청
    수정 후 [재제출] 버튼을 클릭하세요.
