                              재택근무지침
================================================================================

1. 개요
-------
본 지침은 재택근무의 신청, 승인, 근무 방법, 보안 준수사항을 안내합니다.

================================================================================

2. 재택근무 대상
---------------

2.1 신청 자격
✓ 입사 6개월 이상 경과
✓ 최근 평가등급 B등급 이상
✓ 재택근무 가능 업무 담당
✓ 부서장 승인

2.2 재택근무 제외 업무
✗ 고객 대면 업무 (영업, 창구)
✗ 보안 업무 (전산실 관리 등)
✗ 물리적 장비 운영 업무
✗ 기밀 문서 취급 업무 (1등급, 2등급)

================================================================================

3. 재택근무 신청
---------------

3.1 신청 방법
- 사내 포털 > 인사 > 재택근무 신청
- 또는 HR시스템 > 근태 > 재택근무

3.2 신청 기한
- 정기 재택: 최소 1주일 전 신청
- 임시 재택: 전일 17시까지 신청
- 긴급 재택: 당일 09시까지 신청 (사유 필수)

3.3 신청 유형
| 유형 | 설명 | 한도 |
|------|------|------|
| 정기 재택 | 요일 고정 (예: 매주 화/목) | 주 2일 |
| 임시 재택 | 상황에 따른 비정기 신청 | 월 4일 |
| 긴급 재택 | 당일 불가피한 상황 | 월 2일 |

3.4 승인 체계
- 1~2일: 팀장 승인
- 3일 이상 연속: 부서장 승인
- 1주일 이상: 본부장 승인

================================================================================

4. 근무 시간
-----------

4.1 기본 원칙
- 재택근무도 정규 근무시간 준수 (09:00~18:00)
- 점심시간: 12:00~13:00

4.2 유연근무 적용
- 시차출퇴근: 07:00~10:00 출근, 16:00~19:00 퇴근
- 코어타임 필수 근무: 10:00~15:00

4.3 근무 상태 관리
- 출근 체크: 09:00까지 HR시스템 체크인
- 퇴근 체크: 업무 종료 시 체크아웃
- 점심시간 외 30분 이상 자리 비움 시 상태 변경

================================================================================

5. 업무 환경
-----------

5.1 업무 장소
✓ 본인 자택
✓ 승인받은 원격근무지 (부모님 댁 등)
✗ 카페, 도서관 (보안 사유)
✗ 해외 (별도 승인 필요)

5.2 장비
- 회사 지급 노트북 사용 (원칙)
- 개인 PC 사용 시 보안 프로그램 설치 필수
- 모니터, 키보드 등 주변기기: 개인 준비

5.3 네트워크
- 개인 인터넷 사용 (비용 미지원)
- VPN 접속 필수
- 공용 와이파이 사용 금지

================================================================================

6. 커뮤니케이션
--------------

6.1 연락 가능 상태 유지
- 근무시간 중 연락 가능 상태 유지
- 메신저 상태: "재택근무 중"으로 설정
- 전화, 메신저 응답: 10분 이내

6.2 회의 참석
- 화상회의: Zoom, Teams 등 회사 승인 도구
- 카메라 ON 권장 (필수 아님)
- 배경 설정: 가상 배경 또는 적절한 환경

6.3 일일 보고
- 업무 시작 시: 오늘 할 일 공유
- 업무 종료 시: 완료 사항 보고
- 방식: 팀 메신저 채널 또는 이메일

================================================================================

7. 보안 준수사항
---------------

7.1 VPN 사용
- 사내 시스템 접속 시 VPN 필수
- VPN 미접속 시 업무 시스템 접근 불가
- VPN 연결 상태 유지

7.2 정보 보호
✗ 화면 캡처 금지 (DLP 설치 시 자동 차단)
✗ 문서 로컬 저장 금지 (승인 시 제외)
✗ 출력 금지 (재택근무 시)
✗ 외부인 화면 노출 금지

7.3 기기 보안
- 화면잠금: 5분 이내 자동 설정
- 비밀번호: 보안 정책 준수
- 백신: 최신 업데이트 유지

7.4 보안사고 시
- 즉시 IT헬프데스크 또는 보안팀 보고
- 기기 네트워크 연결 해제
- 상세 경위 보고서 작성

================================================================================

8. 근태 관리
-----------

8.1 출퇴근 기록
- HR시스템 통해 체크인/체크아웃
- GPS 정보 수집 없음 (자택 확인 불요)
- 로그 기록으로 근무시간 인정

8.2 초과근무
- 재택근무 시에도 초과근무 신청 가능
- 사전 승인 필요 (팀장)
- 초과근무 시간 기록

8.3 휴가
- 재택근무일에 휴가 사용 가능
- 반차(오전/오후) 사용 가능

================================================================================

9. 평가 및 모니터링
------------------

9.1 성과 평가
- 재택근무도 동일한 성과 기준 적용
- 근무 장소가 아닌 성과로 평가

9.2 모니터링
- 업무 시스템 접속 로그 확인
- VPN 접속 시간 확인
- 불필요한 감시 없음 (신뢰 기반)

9.3 재택근무 제한
다음 경우 재택근무가 제한될 수 있습니다:
- 연락 두절 반복
- 업무 성과 저하
- 보안 위반
- 부서장 판단

================================================================================

10. 비용 지원
------------

10.1 지원 항목
- 현재 별도 비용 지원 없음
- 통신비, 전기료 개인 부담

10.2 장비 지원
- 회사 노트북 제공
- 추가 장비 필요 시 IT팀에 대여 신청

11. 자주 묻는 질문 (FAQ)
-----------------------

Q1. 주 3일 이상 재택근무 가능한가요?
A1. 원칙적으로 주 2일까지 가능합니다.
    특별한 사유(임신, 육아, 간병 등)가 있는 경우
    부서장과 협의하여 추가 승인받을 수 있습니다.

Q2. 카페에서 일해도 되나요?
A2. 보안 사유로 권장하지 않습니다.
    공용 와이파이 사용 금지, 화면 노출 위험 등
    자택에서 근무하시기 바랍니다.

Q3. 재택근무 중 외출해도 되나요?
A3. 점심시간 외에는 원칙적으로 자제해주세요.
    불가피한 외출(병원 등)은 시간 단위 연차를 사용하거나
    업무 조정 후 사후 만회 근무하시면 됩니다.

Q4. 자녀 돌봄과 재택근무를 병행해도 되나요?
A4. 재택근무는 근무이므로, 업무에 집중하셔야 합니다.
    돌봄이 필요한 경우 육아휴직, 가족돌봄휴가 등
    별도 제도를 활용해주세요.

Q5. 해외에서 재택근무 가능한가요?
A5. 원칙적으로 불가합니다.
    특별한 사유가 있는 경우 인사팀과 협의하세요.
    세금, 보험, 보안 이슈가 있어 사전 검토가 필요합니다.
