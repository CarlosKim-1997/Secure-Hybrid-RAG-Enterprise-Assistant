                        IT 헬프데스크 이용 매뉴얼
================================================================================

1. 개요
-------
IT헬프데스크는 임직원의 IT 관련 문의 및 장애를 지원하는 서비스입니다.
본 매뉴얼은 헬프데스크 이용 방법을 안내합니다.

================================================================================

2. 연락 방법
-----------

2.1 연락처
┌─────────────────────────────────────────────────────┐
│  📞 전화: 내선 9999 (외부: 02-XXXX-9999)            │
│  📧 이메일: helpdesk@company.com                   │
│  💻 웹: 사내 포털 > IT지원 > 서비스 요청            │
│  💬 메신저: Teams "IT헬프데스크" 채널              │
└─────────────────────────────────────────────────────┘

2.2 운영 시간
- 평일: 08:30 ~ 18:30
- 점심: 12:00 ~ 13:00 (긴급 건만 대응)
- 야간/주말: 긴급 장애 핫라인 (010-XXXX-XXXX)

2.3 긴급 연락 기준
다음 경우 야간/주말 긴급 연락 가능:
- 서비스 전면 장애
- 보안 사고 의심
- 중요 업무 불가 (다수 영향)

================================================================================

3. 서비스 요청 방법
------------------

3.1 웹 신청 (권장)
① 사내 포털 접속
② [IT지원] > [서비스 요청] 클릭
③ 요청 유형 선택
④ 상세 내용 작성
⑤ [신청] 버튼 클릭
⑥ 접수 번호 확인

3.2 전화 신청
① 내선 9999 전화
② 사번, 이름 안내
③ 문의 내용 설명
④ 접수 번호 안내 받음

3.3 이메일 신청
수신: helpdesk@company.com
제목: [요청유형] 간단한 설명
내용:
- 이름/사번/연락처
- 문의 내용 상세
- 오류 메시지 (있는 경우)
- 긴급도 (긴급/일반)

================================================================================

4. 서비스 유형
-------------

4.1 장애 신고
- PC 부팅 안됨
- 네트워크 연결 안됨
- 프로그램 오류
- 프린터 출력 안됨

4.2 서비스 요청
- 소프트웨어 설치 요청
- 계정/권한 신청
- 장비 대여 요청
- VPN 설정 요청

4.3 문의
- 사용 방법 문의
- 정책 문의
- 기타 IT 관련 질문

================================================================================

5. 처리 절차
-----------

5.1 처리 단계
① 접수: 요청 접수 및 접수번호 발급
② 분류: 유형/긴급도 분류
③ 할당: 담당자 배정
④ 처리: 문제 해결
⑤ 완료: 결과 안내 및 종료

5.2 처리 시간 (SLA)
| 긴급도 | 최초 응답 | 해결 목표 |
|--------|-----------|-----------|
| 긴급 | 30분 이내 | 4시간 이내 |
| 높음 | 2시간 이내 | 8시간 이내 |
| 보통 | 4시간 이내 | 24시간 이내 |
| 낮음 | 8시간 이내 | 72시간 이내 |

5.3 진행 상황 확인
- 사내 포털 > IT지원 > 내 요청 현황
- 접수번호로 조회

6. 자주 묻는 질문 (FAQ)
----------------------

【계정/비밀번호】

Q1. 비밀번호를 잊어버렸어요.
A1. 두 가지 방법이 있습니다:
    ① 셀프 서비스: 로그인 화면 > "비밀번호 찾기"
       - 등록된 휴대폰으로 인증
    ② 헬프데스크 연락: 본인 확인 후 임시 비밀번호 발급

Q2. 비밀번호 변경 방법은?
A2. Ctrl + Alt + Delete > "비밀번호 변경"
    또는 사내 포털 > 마이페이지 > 비밀번호 변경

Q3. 계정이 잠겼어요.
A3. 5회 연속 비밀번호 오류 시 자동 잠금됩니다.
    헬프데스크 연락 시 본인 확인 후 해제해 드립니다.

【PC/노트북】

Q4. PC가 느려요.
A4. 다음을 먼저 시도해 보세요:
    ① 재부팅 (일주일에 1회 권장)
    ② 불필요한 프로그램 종료
    ③ 디스크 정리 (시작 > 디스크 정리)
    개선되지 않으면 헬프데스크로 연락주세요.

Q5. 새 PC/노트북을 받고 싶어요.
A5. 교체 주기:
    - 노트북: 4년
    - 데스크탑: 5년
    기준 미달 시 부서장 승인 후 신청 가능합니다.
    사내 포털 > IT지원 > 장비 교체 신청

Q6. 개인 노트북을 업무에 사용해도 되나요?
A6. 원칙적으로 불가합니다.
    부득이한 경우 BYOD 신청 후 보안 프로그램 설치 조건으로 
    승인받을 수 있습니다 (보안팀 협의 필요).

【소프트웨어】

Q7. 프로그램을 설치하고 싶어요.
A7. 승인된 소프트웨어만 설치 가능합니다.
    사내 포털 > IT지원 > 소프트웨어 설치 신청
    - 무료 소프트웨어: 팀장 승인
    - 유료 소프트웨어: 부서장 승인 + 예산 확보

Q8. Office가 안 열려요.
A8. 다음을 시도해 보세요:
    ① Office 프로그램 재시작
    ② PC 재부팅
    ③ Office 복구 (제어판 > 프로그램 > Office > 복구)
    해결 안 되면 헬프데스크 연락주세요.

【네트워크/인터넷】

Q9. 인터넷이 안 돼요.
A9. 확인 순서:
    ① 랜선 연결 상태 확인
    ② PC 재부팅
    ③ 다른 PC도 동일한지 확인
    - 본인 PC만: 헬프데스크 연락
    - 여러 PC: 네트워크 장애, 긴급 신고

Q10. VPN 연결이 안 돼요.
A10. 확인 사항:
    ① 인터넷 연결 정상 여부
    ② VPN 프로그램 버전 (최신 버전 권장)
    ③ 아이디/비밀번호 정확성
    ④ OTP 정상 여부
    해결 안 되면 헬프데스크 연락주세요.

【프린터】

Q11. 프린터 출력이 안 돼요.
A11. 확인 순서:
    ① 프린터 전원/용지 확인
    ② 프린터 선택 확인 (기본 프린터)
    ③ 인쇄 대기열 확인 (취소 후 재시도)
    해결 안 되면 헬프데스크 연락주세요.

Q12. 프린터를 추가하고 싶어요.
A12. 사내 포털 > IT지원 > 프린터 설정 요청
    또는 헬프데스크에 프린터명/위치 알려주시면 
    원격으로 설정해 드립니다.

【이메일】

Q13. 이메일이 안 보내져요.
A13. 확인 사항:
    ① 첨부파일 크기 (10MB 제한)
    ② 수신자 주소 정확성
    ③ 네트워크 연결 상태
    대용량 파일은 사내 파일 공유 서비스를 이용하세요.

Q14. 스팸 메일이 많이 와요.
A14. 스팸 메일을 선택 > "정크메일 보고" 하시면 
    학습되어 차단됩니다.
    피싱 의심 메일은 보안팀에 신고해 주세요.

================================================================================

7. 원격 지원
-----------

7.1 원격 지원 안내
- 복잡한 문제는 원격으로 PC를 지원합니다
- 원격 지원 동의 후 진행

7.2 원격 지원 절차
① 헬프데스크 연락
② 원격 지원 필요 시 안내
③ 원격 프로그램 실행 (바탕화면 "원격지원" 아이콘)
④ 안내받은 접속 코드 입력
⑤ 원격 연결 승인
⑥ 담당자가 원격으로 문제 해결

================================================================================

8. 장비 대여
-----------

8.1 대여 가능 장비
- 노트북 (회의, 출장용)
- 빔프로젝터
- 화상회의 장비
- 외장하드 (보안USB)

8.2 대여 신청
- 사내 포털 > IT지원 > 장비 대여
- 대여 기간, 용도 입력
- 팀장 승인

8.3 대여 기간
- 일반: 최대 1주일
- 장기: 부서장 승인 시 연장 가능

8.4 반납
- 기한 내 IT자산실 반납
- 미반납 시 독촉 → 부서장 통보

================================================================================

9. 긴급 상황 대처
----------------

9.1 PC 바이러스/랜섬웨어 의심
① 즉시 랜선 분리 (또는 와이파이 끄기)
② 전원 끄지 않기
③ 보안팀 즉시 연락 (내선 5000)

9.2 데이터 삭제/손실
① 복구 시도하지 않기
② 헬프데스크 즉시 연락
③ 백업 여부 확인

9.3 장비 분실/도난
① 즉시 헬프데스크 신고
② 보안팀 연락
③ 분실 경위서 작성

================================================================================

10. 자주 사용하는 단축키
-----------------------

【Windows】
- Ctrl+Alt+Del: 작업관리자, 비밀번호 변경
- Win+L: 화면 잠금
- Win+D: 바탕화면 보기
- Alt+Tab: 창 전환
- Ctrl+Shift+Esc: 작업 관리자

【Office 공통】
- Ctrl+S: 저장
- Ctrl+Z: 실행 취소
- Ctrl+C/V/X: 복사/붙여넣기/잘라내기
- Ctrl+F: 찾기

---------------
