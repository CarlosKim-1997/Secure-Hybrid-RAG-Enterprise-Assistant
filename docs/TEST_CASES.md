# 금융사 내규 AI Assistant — 테스트 케이스

본 테스트 케이스는 **실제 내규 문서(data/internal_docs)에 있는 내용만으로 답할 수 있는 질문**과, **문서에 없는 정보를 묻거나 엉뚱한 질문**을 모두 포함합니다.  
시스템이 “해당 정보를 찾을 수 없습니다” 또는 “제공된 문서만으로는 부족합니다”라고 적절히 답하는지도 함께 검증합니다.

---

## 사용 방법

- 각 질문을 앱에 입력한 뒤 **예상 유형**(답 가능 / 정보 없음)과 **예상 답변 요지**를 참고해 비교하세요.
- **검증 포인트**: Query Planner 분석, 검색된 문서 목록, 답변의 문서 인용(인용 가능한 문서 ID만 사용), 한국어만 사용, 정보 없을 때의 거절 답변.

---

# A. 문서에서 답할 수 있는 질문

아래 질문은 내규 문서에 실제로 존재하는 내용으로 답변이 가능해야 합니다.  
답변에는 **반드시 인용 가능한 문서 ID**(검색된 문서)만 사용하고, 문서에 없는 조항·규정을 만들면 안 됩니다.

---

## A-1. 해외 출장 + 고객 접대비 150만원 정산

**입력 질문**
```
해외 출장 가서 현지 고객사 접대비 150만원 썼는데 어떻게 정산해요?
```

**문서 출처**  
출장관리규정(FIN-REG-002), 접대비관리규정(FIN-REG-004), 해외출장지침(FIN-GUI-004), 경비관리규정(FIN-REG-001), 양식 카탈로그(FIN-F-012, ADM-F-001 등)

**예상 답변 요지**
- 해외 출장 + 접대비 결합 케이스로, FIN-REG-002, FIN-REG-004, FIN-GUI-004 적용
- 150만원 구간: 100만원 이상 300만원 미만 → CFO 사전 승인(접대비규정 제8조)
- 처리 절차: 경비정산 유형 선택, 접대 목적·참석자 기재, 증빙 첨부, 승인 요청
- 필요 양식: FIN-F-012 해외접대비 정산서, FIN-F-002 출장정산서
- 인용 문서: FIN-REG-002, FIN-REG-004, FIN-GUI-004 등 검색된 문서 ID만 사용

---

## A-2. 연차휴가 신청 방법

**입력 질문**
```
연차휴가 신청은 어떻게 하나요?
```

**문서 출처**  
휴가규정(HR-REG-003), 휴가신청지침(HR-GUI-005), 양식 HR-F-001 휴가신청서

**예상 답변 요지**
- 신청 시기: 1일 1영업일 전, 3일 이상 3영업일 전, 5일 이상 7영업일 전(휴가규정 제6조)
- 신청 방법: HR시스템 전자결재
- 승인권자: 3일 이내 팀장, 3일 초과 7일 이내 부서장, 7일 초과 본부장(제7조)
- 반일·시간 단위 사용 가능
- 인용 문서: HR-REG-003 등 검색된 문서만

---

## A-3. 협력업체 직원 시스템 접근권한 부여

**입력 질문**
```
협력업체 직원에게 시스템 접근권한 부여하려면 어떻게 해요?
```

**문서 출처**  
접근통제규정(SEC-REG-001), 외주관리규정(VND-REG-001), 양식 SEC-F-003 외부인 시스템 접근 신청서, VND-F-002 협력업체 보안 서약서

**예상 답변 요지**
- 협력업체 계정: 접근통제규정 제7조·제8조(계정 생성 절차, 협력업체 명명 규칙·기간)
- 절차: 부서장 접근권한 신청서 작성 → 정보보안 담당 검토 → 계정 생성 → 신청자 전달
- 외주관리규정: 협력업체 인력 투입(제12조), 접근권한 관리(제13조), 보안서약서 징구
- 사용 기간: 프로젝트 종료일, 연장 시 재승인
- 인용 문서: SEC-REG-001, VND-REG-001 등만 사용(계정관리매뉴얼 SEC-MAN-001은 현재 없음)

---

## A-4. 보안사고 발생 시 대응

**입력 질문**
```
보안사고 발생 시 어떻게 대응해야 하나요?
```

**문서 출처**  
보안사고대응규정(SEC-REG-003), 침해사고대응매뉴얼(SEC-MAN-002)

**예상 답변 요지**
- 사고 등급(1~4급) 및 대응 시간(즉시, 30분 내, 4시간 내, 익영업일)
- 신고: 정보보호본부(보안사고대응팀/CERT) 24시간 연락
- 1급·2급: CISO 최종 판정
- 침해사고대응매뉴얼: 등급별 즉시 조치, 격리, 증거 보존
- 인용 문서: SEC-REG-003, SEC-MAN-002만 사용

---

## A-5. 법인카드 한도 증액 신청

**입력 질문**
```
법인카드 한도 증액 신청 방법이 궁금해요.
```

**문서 출처**  
경비관리규정(FIN-REG-001), 법인카드사용지침(FIN-GUI-002), 양식 FIN-F-006 법인카드 한도 변경 신청서

**예상 답변 요지**
- 신청 절차: 법인카드 한도 변경 신청서 작성 → 증액 사유 기재 → 부서장 승인 후 재무팀 제출 → 검토 후 카드사 한도 변경
- 필요 양식: FIN-F-006
- 담당: 재무기획팀
- 인용 문서: FIN-REG-001, FIN-GUI-002 등만 사용

---

## A-6. 재택근무 시 보안 준수사항

**입력 질문**
```
재택근무 시 보안 준수사항이 뭐예요?
```

**문서 출처**  
재택근무지침(HR-GUI-010)

**예상 답변 요지**
- VPN 필수, 미접속 시 업무 시스템 접근 불가
- 화면 캡처 금지, 문서 로컬 저장·출력 금지, 외부인 화면 노출 금지
- 화면잠금 5분 이내, 비밀번호·백신 정책 준수
- 보안사고 시 즉시 IT헬프데스크·보안팀 보고, 기기 연결 해제, 경위 보고서 작성
- 인용 문서: HR-GUI-010만 사용

- 인용 문서: HR-REG-003만 사용

해외 출장 신청은 며칠 전에 해야 하나요?

출장관리규정(FIN-REG-002) 제6조, 해외출장지침(FIN-GUI-004)

**예상 답변 요지**

- 국내 당일 1일 전, 국내 숙박 3일 전
- 인용 문서: FIN-REG-002, FIN-GUI-004만 사용

# B. 문서에 없는 정보 또는 엉뚱한 질문

아래 질문은 **내규 문서에 해당 내용이 없거나**, 주제가 문서 범위를 벗어납니다.  
시스템은 **해당 정보를 찾을 수 없다**고 안내하거나, **제공된 문서만으로는 부족하다**·**담당부서 문의**를 안내해야 합니다.  
**문서에 없는 규정·조항·문서 ID를 만들어 내면 안 됩니다.**

---

## B-1. 직원 단체 보험 (문서에 없음)

**입력 질문**
```
직원 단체 보험 가입 조건이 어떻게 되나요?
```

**예상 답변 요지**
- 제공된 내규 문서에 단체 보험 가입 조건에 대한 내용이 없음
- "제공된 문서만으로는 해당 정보를 찾을 수 없습니다. 인사팀 또는 복리후생 담당부서에 문의하세요." 등으로 안내
- **있으면 안 되는 것**: 존재하지 않는 문서 ID나 조항을 인용

---

## B-2. 회의실 예약 취소 (문서에 없음)

**입력 질문**
```
회의실 예약 취소는 이용 몇 시간 전까지 해야 하나요?
```

**예상 답변 요지**
- 양식 카탈로그에 회의실 예약 신청서(ADM-F-003)는 있으나, **취소 시한(몇 시간 전)**은 문서에 없음
- "제공된 문서에는 회의실 예약 취소 시한이 기재되어 있지 않습니다. 총무팀에 문의하세요." 등
- **있으면 안 되는 것**: 취소 시한을 임의로 숫자로 만들어 내는 것

---

## B-3. AI 활용 가이드라인 (문서에 없음)

**입력 질문**
```
AI 활용 업무 가이드라인이 있나요? 어디서 보나요?
```

**예상 답변 요지**
- 현재 내규 문서 목록에 AI 활용·생성형 AI 관련 가이드라인 없음
- "제공된 문서에서는 AI 활용 업무 가이드라인을 찾을 수 없습니다. IT본부 또는 준법담당부서에 문의하세요." 등
- **있으면 안 되는 것**: 없는 가이드라인이나 문서 ID를 만들어 인용

---

## B-4. 재택근무 통신비 지원 (문서에 없음)

**입력 질문**
```
재택근무 시 통신비 지원 금액은 얼마인가요?
```

**예상 답변 요지**
- 재택근무지침(HR-GUI-010)에는 "개인 인터넷 사용 (비용 미지원)"으로 되어 있음
- 통신비 지원 금액을 구체적으로 적어 둔 조항은 없을 수 있음 → "비용 미지원"이면 그에 맞게 답하고, 지원 금액 문의는 인사팀 안내
- **있으면 안 되는 것**: 지원 금액(숫자)을 문서에 없는데 만들어 내는 것

---

## B-5. 회사 기부금 한도 (문서에 없음)

**입력 질문**
```
회사 기부금 한도는 얼마인가요?
```

**예상 답변 요지**
- 제공된 내규 문서에 기부금 한도에 대한 규정이 없음
- "제공된 문서만으로는 해당 정보를 찾을 수 없습니다. 재무팀 또는 경영기획부에 문의하세요." 등
- **있으면 안 되는 것**: 존재하지 않는 규정·한도 금액을 만들어 인용
